Script Name:
Torn Gym Energy Calculator

Description
Maximize your gym progress with this intelligent, real-time energy calculator. Designed to look and feel like a native part of the Torn UI, this script automatically calculates exactly how much energy, time, and effort is required to unlock your next gym.

Key Features:

Real-Time Tracking: Updates your "Remaining Energy" instantly as you train—no page refresh required!

Smart Range Estimates: Calculates the Minimum and Maximum energy required based on Torn's rounded % system, giving you the most accurate data possible.

API Integrated: Automatically detects your Music Store (30% boost) perk and daily energy regeneration rate using your Public API Key.

Native UI: Features a sleek, symmetrical Teal/Cyan progress bar that fits perfectly into Torn’s dark mode.

Persistent: Works seamlessly even when switching between different gyms without needing to reload.


How to Use
Install the Script: Add the script to Tampermonkey (or Violentmonkey).
Go to the Gym: Navigate to torn.com/gym.php. You will see a new Teal Bar appear below your current gym's unlocked list.

Enter API Key:
Click the small Gear Icon (⚙) on the right side of the bar.

Enter your Public API Key (found in Torn Settings).
Click Save.

Refresh: Refresh the page once to let the script fetch your perks and stats.

Train: As you use energy training, the "Remaining E" numbers will drop instantly to show your progress!
